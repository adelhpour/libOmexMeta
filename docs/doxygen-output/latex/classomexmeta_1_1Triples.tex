\doxysubsection{Detailed Description}
\begin{DoxyVerb}@brief A Triples object is a collection of Triple objects.
\end{DoxyVerb}
  the implementation of \mbox{\hyperlink{classomexmeta_1_1Triples}{Triples}} is hampered by the inability to clone \mbox{\hyperlink{classomexmeta_1_1Triple}{Triple}} objects. Instead triples must use move semantics to get \mbox{\hyperlink{classomexmeta_1_1Triples}{Triples}} in or out of a \mbox{\hyperlink{classomexmeta_1_1Triples}{Triples}} object. Under the hood, the \mbox{\hyperlink{classomexmeta_1_1Triples}{Triples}} object is just a std\+::vector$<$\+Triple$>$.
