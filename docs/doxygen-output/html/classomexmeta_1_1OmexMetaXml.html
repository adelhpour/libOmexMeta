const std::string & omexmeta::OmexMetaXml::getMetaidBase ( ) const

get the base for the metaid used by the current xml type. This base is used as a precursor string for generating new metaids if requested by the user by setting generate_new_metaids=true.

the base is given as argument to the OmexMetaXmlAssistant on instantiation. If base is "metaid" then ids generated will be metaid_0000
