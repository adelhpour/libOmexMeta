◆ free()

void omexmeta::PhysicalForce::free ( )

Free nodes associated with PhysicalForce.

The PhysicalForce is owned by the caller if the PhysicalForce::toTriples method is NOT used. When toTriples() is used, the nodes that create the PhysicalForce are "given" to the Triples object, which automatically destroys them at the right time.

Most of the time, users will not have to remember to free the PhysicalForce themselves as the toTriples method is always used.
