Detailed Description

PhysicalProperty is a subclass of the Resource

object. It is a special type of Resource that is restricted to terms from the ontology of physics for biology.

Usage: PhysicalProperty r("http://identifiers.org/opb:opb12345"); is equilavent to: PhysicalProperty r("opb:opb12345"); is equilavent to: PhysicalProperty r("opb/opb12345");
