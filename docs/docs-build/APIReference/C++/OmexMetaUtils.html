std::string prepareBaseUri(std::string str, bool absolute_path = false)¶

process a string intended to be a base uri.

Base uri’s are important in redland libraries. If they do not begin with

http or https, sparql querying will break down. This is a helper function to ensure the base uri is properly formatted.
Parameters

  • str: The string that will become a base uri

  • absolute_path: automatically make str an absolute path to the current working directory.
